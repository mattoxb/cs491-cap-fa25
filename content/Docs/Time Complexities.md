# Time Complexities

For a modern computer, here are the largest sample sizes that can be run in one second for a given time complexity.

| $n$             | Time Complexity                         |
|-----------------|-----------------------------------------|
| $10$            | ${\cal O}(n!)$                          |
| $25$            | ${\cal O}(2^n)$                         |
| $300$           | ${\cal O}(n^3)$                         |
| $5,000$         | ${\cal O}(n^2)$                         |
| $50,000$        | ${\cal O}(n^{1.5})$                     |
| $10^5$          | ${\cal O}(n\times \log n\times \log n)$ |
| $5 \times 10^5$ | ${\cal O}(n\times \log n)$              |
| $10^7$          | ${\cal O}(n)$                           |
